01 Project Summary

A brief executive summary of what Premier Shop is and what it does.

Premier Shop is a full-stack, production-grade e-commerce web application built for premium retail and wholesale grocery operations. It provides a complete shopping experience including product browsing, cart management, secure checkout with coupon and loyalty rewards, order tracking with driver delivery, and a powerful admin dashboard with analytics and reporting.

Key Capabilities

  • Multi-Role System — Admin, Customer, and Driver roles with granular RBAC permissions.
  • Age-Restricted Products — Automatic enforcement blocking under-16 users from viewing or purchasing restricted items (alcohol, tobacco).
  • Dynamic Shipping Calculator — Google Maps Distance Matrix API integration for real-time distance-based delivery pricing.
  • Loyalty & Rewards Program — Earn/redeem points on purchases, with clawback on cancellation.
  • QR Code Scanning — Admin barcode scanner for rapid product lookup and stock management.
  • PDF Invoice Generation — DomPDF-powered downloadable sales reports and order receipts.
  • OTP Email Verification — 6-digit code verification during registration with rate-limited retries.
  • Real-Time Notifications — In-app bell notifications for order updates, low stock, and return requests.
  • Driver Delivery Portal — Dedicated driver dashboard with duty toggling, order viewing, and photo delivery proof upload.

09 Services

Business logic services extracted from controllers.

ShippingService

Method: calculateDrivingDistance(string $origin, string $destination): ?float

Integrates with the Google Maps Distance Matrix API to calculate the driving distance (in kilometres) between the shop's origin address and the customer's delivery address. Used by the checkout controller to determine:

  • Free delivery if the order exceeds the free delivery threshold OR the distance is within the free delivery radius.
  • Surcharge pricing calculated as: flat_rate_fee + (extra_miles × surcharge_per_mile) for distances beyond the free radius.

Falls back to flat-rate shipping if the Google Maps API key is not configured or the API request fails.

10 Middleware & Security

Security layers and request filters protecting the application.

SecurityHeadersMiddleware

Injects the following security headers on every HTTP response:

  • X-Frame-Options: SAMEORIGIN — Prevents clickjacking by blocking iframe embedding from external domains.
  • X-Content-Type-Options: nosniff — Prevents MIME type sniffing attacks.
  • X-XSS-Protection: 1; mode=block — Enables browser-level XSS filtering.
  • Referrer-Policy: strict-origin-when-cross-origin — Controls referrer header leakage.
  • Permissions-Policy — Disables geolocation, camera, microphone, and gyroscope access.
  • Strict-Transport-Security — Enforces HTTPS for 1 year on secure connections.
  • Content-Security-Policy — Strict allowlist for script/style/font/image sources.

Additional Security Measures

  • CSRF Protection — All POST/PATCH/DELETE forms include @csrf tokens.
  • Rate Limiting — Checkout and OTP verification endpoints are throttled to prevent brute-force attacks.
  • Password Hashing — All passwords stored using Laravel's Hash::make() (bcrypt).
  • Prunable Users — Unverified accounts older than 24 hours are automatically prunable.
  • Authorization Guards — Driver and admin controllers enforce ownership checks before data access.
